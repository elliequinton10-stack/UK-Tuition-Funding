## 🚀 How to Deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `baytkay-blessing`)
2. Upload all four files:
   - `index.html` *(rename `blessing-landing.html` → `index.html`)*
   - `style.css`
   - `script.js`
   - `logo.png`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save**
6. Your site will be live at: `https://yourusername.github.io/baytkay-blessing/`
